### Сервисы (src/services/)

#### api.ts
**Размер**: ~3 KB  
**Строк**: ~90  
**Назначение**: HTTP API клиент

**Методы**:
- `setToken()` - установка токена
- `request()` - базовый запрос
- `get()` - GET запрос
- `post()` - POST запрос
- `put()` - PUT запрос
- `delete()` - DELETE запрос
- `uploadFile()` - загрузка файлов с прогрессом

**Особенности**:
- Автоматическая установка headers
- Обработка ошибок
- JSON парсинг
- XMLHttpRequest для uploads

---

#### websocket.ts
**Размер**: ~3 KB  
**Строк**: ~110  
**Назначение**: WebSocket клиент

**Методы**:
- `constructor()` - инициализация
- `on()` - подписка на событие
- `off()` - отписка
- `send()` - отправка сообщения
- `disconnect()` - закрытие соединения

**Особенности**:
- Автоматическое переподключение
- Event-based архитектура
- Exponential backoff
- Wildcard handlers
