# Руководство пользователя

## 🚀 Начало работы

### Регистрация

1. Откройте приложение в браузере
2. На экране входа нажмите "Нет аккаунта? Зарегистрируйтесь"
3. Заполните форму:
   - **Имя** - ваше отображаемое имя
   - **Имя пользователя** - уникальный @username
   - **Email** - ваш email адрес
   - **Пароль** - надежный пароль
4. Нажмите "Зарегистрироваться"

### Вход

1. Введите имя пользователя
2. Введите пароль
3. Нажмите "Войти"

## 👤 Настройка профиля

### Открытие профиля
- Кликните на свой аватар в левом верхнем углу

### Редактирование профиля
1. В окне профиля нажмите "Редактировать профиль"
2. Измените нужные данные:
   - **Имя** - ваше имя
   - **Имя пользователя** - @username
   - **О себе** - краткая биография
3. Нажмите "Сохранить"

### Изменение аватара
1. В режиме редактирования нажмите на иконку камеры на аватаре
2. Выберите фото с компьютера
3. Превью покажет новый аватар
4. Нажмите "Сохранить"

## 💬 Работа с чатами

### Создание нового чата

1. Нажмите иконку ✏️ (редактирование) в верхней панели
2. В поле поиска введите имя или @username пользователя
3. Кликните на нужного пользователя
4. Чат будет создан автоматически

### Поиск чатов

1. Используйте поле поиска в верхней части боковой панели
2. Введите название чата или имя пользователя
3. Результаты появятся автоматически

### Фильтрация чатов

Используйте вкладки для фильтрации:
- **Все** - показывает все чаты и каналы
- **Чаты** - только личные переписки
- **Каналы** - только каналы

## 📝 Отправка сообщений

### Текстовое сообщение

1. Выберите чат из списка
2. Введите текст в поле внизу
3. Нажмите Enter или кнопку отправки ✈️

**Совет**: Shift + Enter для новой строки

### Эмодзи

1. Нажмите иконку смайлика 😊 рядом с полем ввода
2. Выберите нужный эмодзи из палитры
3. Эмодзи вставится в текст

### Отправка файлов

#### Фото и документы
1. Нажмите иконку скрепки 📎
2. Выберите файлы на компьютере
3. Превью появится над полем ввода
4. При необходимости добавьте текст
5. Нажмите отправить

#### Поддерживаемые типы
- Изображения: JPG, PNG, GIF
- Видео: MP4, MOV, AVI
- Документы: PDF, DOC, DOCX, XLS, XLSX

### Реакции на сообщения

1. Наведите курсор на сообщение
2. Появятся быстрые реакции: 👍 ❤️ 😂 😮 😢 🔥
3. Кликните на нужную реакцию
4. Реакция появится под сообщением

## 📢 Каналы

### Создание канала

1. Нажмите иконку мегафона 📢 в верхней панели
2. Введите название канала
3. (Опционально) Добавьте описание
4. Нажмите "Создать"

### Особенности каналов

- В каналах могут писать только администраторы
- Показывается количество подписчиков
- Нет индикатора "онлайн/офлайн"
- Нет звонков

## 📞 Звонки

### Аудиозвонок

1. Откройте чат с пользователем
2. Нажмите иконку телефона 📱 в шапке чата
3. Дождитесь ответа собеседника
4. Для завершения нажмите красную кнопку

### Видеозвонок

1. Откройте чат с пользователем
2. Нажмите иконку видео 🎥 в шапке чата
3. Дождитесь ответа
4. Ваше видео будет в углу, видео собеседника на весь экран

### Управление во время звонка

#### Аудио
- 🎤 **Микрофон** - включить/выключить звук
- ⚙️ **Настройки** - выбор устройств
- 📞 **Завершить** - закончить звонок

#### Видео
- 🎥 **Камера** - включить/выключить видео
- 🎤 **Микрофон** - включить/выключить звук
- ⚙️ **Настройки** - выбор устройств
- ⛶ **Полный экран** - развернуть на весь экран
- 📞 **Завершить** - закончить звонок

## 🎨 Настройки интерфейса

### Переключение темы

1. Нажмите иконку 🌙/☀️ в верхней панели
2. Тема изменится автоматически
3. Настройка сохраняется в браузере

**Доступные темы**:
- 🌙 Темная (по умолчанию)
- ☀️ Светлая

## 🔍 Поиск

### Глобальный поиск

1. Введите запрос в поле поиска боковой панели
2. Появятся результаты:
   - Пользователи
   - Чаты
   - Каналы

### Поиск пользователей

1. Нажмите "Новый чат" ✏️
2. Введите имя или @username
3. Выберите пользователя из результатов

## ⚙️ Дополнительные возможности

### Просмотр информации о чате

В шапке чата отображается:
- Аватар собеседника/канала
- Имя
- Статус онлайн (для личных чатов)
- Количество подписчиков (для каналов)
- Индикатор "печатает..."

### Индикатор "печатает..."

Когда собеседник вводит сообщение, вы увидите:
```
печатает...
```
под именем чата в шапке

### Непрочитанные сообщения

- Красный значок с числом показывает количество непрочитанных
- Появляется справа от названия чата

### Статус отправки

Ваши сообщения имеют индикаторы:
- ✓ Одна галочка - отправлено
- ✓✓ Две галочки - доставлено
- ✓✓ (синие) - прочитано

## 🔒 Безопасность

### Выход из аккаунта

1. Откройте профиль (клик на аватар)
2. Нажмите "Выйти"
3. Вы будете перенаправлены на экран входа

### Безопасность пароля

Рекомендации:
- Используйте уникальный пароль
- Минимум 8 символов
- Сочетание букв, цифр и символов
- Не используйте личную информацию

## 💡 Полезные советы

### Клавиатурные сокращения

- **Enter** - отправить сообщение
- **Shift + Enter** - новая строка
- **Esc** - закрыть модальное окно (в будущем)

### Быстрые действия

- Двойной клик на сообщение - добавить реакцию ❤️ (в будущем)
- Клик правой кнопкой - контекстное меню (в будущем)

### Оптимизация

- Регулярно очищайте кэш браузера
- Используйте стабильное интернет-соединение
- Для звонков рекомендуется проводное подключение

## 🐛 Решение проблем

### Не могу войти
1. Проверьте правильность ввода username и пароля
2. Убедитесь в стабильности интернет-соединения
3. Очистите кэш браузера
4. Попробуйте другой браузер

### Не приходят сообщения
1. Обновите страницу (F5)
2. Проверьте интернет-соединение
3. Убедитесь, что WebSocket подключен

### Не работают звонки
1. Разрешите доступ к камере/микрофону в браузере
2. Проверьте настройки конфиденциальности
3. Используйте HTTPS соединение
4. Попробуйте другой браузер (рекомендуется Chrome)

### Медленно загружается
1. Проверьте скорость интернета
2. Закройте другие вкладки браузера
3. Очистите кэш
4. Используйте последнюю версию браузера

### Не загружаются файлы
1. Проверьте размер файла (макс 10MB)
2. Проверьте формат файла
3. Попробуйте другой файл
4. Проверьте интернет-соединение
