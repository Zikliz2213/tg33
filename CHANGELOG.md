Все важные изменения в проекте документируются в этом файле.

## [1.0.0] - 2024-01-15

### Добавлено ✨

#### Аутентификация
- Форма регистрации с валидацией полей
- Форма входа в систему
- JWT токен аутентификация
- Автоматическое сохранение сессии в localStorage
- Проверка токена при загрузке приложения
- Красивый градиентный экран авторизации

#### Профиль пользователя
- Просмотр профиля (аватар, имя, username, био)
- Редактирование профиля
- Загрузка и изменение аватара с превью
- Кнопка выхода из системы
- Модальное окно профиля

#### Интерфейс мессенджера
- Адаптивная боковая панель с чатами
- Поле поиска по чатам и контактам
- Фильтры: Все, Чаты, Каналы
- Переключатель темы (светлая/темная)
- Индикаторы непрочитанных сообщений
- Статусы онлайн/офлайн

#### Чаты
- Создание личных чатов
- Список чатов с последними сообщениями
- Аватары пользователей
- Время последнего сообщения
- Выделение активного чата
- Модальное окно создания чата

#### Сообщения
- Отправка текстовых сообщений
- Поддержка многострочного текста (Shift+Enter)
- Emoji picker с категориями (80+ эмодзи)
- Отправка файлов (изображения, документы)
- Превью вложенных файлов
- Реакции на сообщения (6 быстрых эмодзи)
- Индикатор "печатает..."
- Время отправки сообщения
- Статус прочтения (двойная галочка)
- Автоматическая прокрутка к новым сообщениям

#### Каналы
- Создание каналов
- Модальное окно создания канала
- Отображение количества подписчиков
- Отдельная иконка для каналов
- Логика разделения чатов и каналов

#### Звонки
- UI для аудиозвонков
- UI для видеозвонков
- Элементы управления:
  - Включение/выключение микрофона
  - Включение/выключение камеры
  - Завершение звонка
  - Настройки
  - Полноэкранный режим
- Таймер продолжительности звонка
- Статусы звонка (calling, connected, ended)
- Интеграция с MediaDevices API
- Picture-in-Picture для видео

#### Real-time функционал
- WebSocket соединение
- Автоматическое переподключение (exponential backoff)
- Обработка событий:
  - Новые сообщения
  - Индикатор печатания
  - Онлайн/офлайн статусы
  - Прочтение сообщений
- Event-based архитектура WebSocket

#### Темы оформления
- Темная тема (по умолчанию)
- Светлая тема
- Плавное переключение тем
- Сохранение выбора в localStorage
- Фирменные цвета Telegram

#### Дизайн и UI
- Максимальное соответствие Telegram Web
- Плавные анимации и переходы
- Характерные "хвостики" у сообщений
- Hover эффекты
- Адаптивный дизайн для всех устройств
- Font Awesome иконки
- Кастомный scrollbar

## Известные проблемы 🐛

### Minor
- WebSocket может не переподключиться после длительного offline
- Emoji picker закрывается только по клику на эмодзи
- Нет виртуализации для длинных списков чатов

### Не реализовано
- Редактирование отправленных сообщений
- Удаление сообщений
- Множественный выбор сообщений
- Drag & drop для файлов
- Копирование/вставка изображений

## Формат версий

Используется [Semantic Versioning](https://semver.org/):
- MAJOR версия для несовместимых изменений API
- MINOR версия для новых функций с обратной совместимостью
- PATCH версия для исправления багов

## Типы изменений

- `Added` - новые функции
- `Changed` - изменения в существующем функционале
- `Deprecated` - функции, которые скоро будут удалены
- `Removed` - удаленные функции
- `Fixed` - исправления багов
- `Security` - исправления уязвимостей
