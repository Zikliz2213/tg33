# Подробное описание функционала

## 📱 Экран авторизации (AuthScreen)

### Возможности
- ✅ Переключение между формами входа и регистрации
- ✅ Валидация всех полей
- ✅ Красивая анимация и градиентный фон
- ✅ Обработка ошибок с понятными сообщениями
- ✅ Индикатор загрузки при отправке запроса
- ✅ Автоматическое сохранение токена в localStorage
- ✅ Адаптивный дизайн для мобильных устройств

### Поля формы

**Регистрация:**
- Имя (displayName)
- Имя пользователя (username)
- Email
- Пароль

**Вход:**
- Имя пользователя (username)
- Пароль

## 🏠 Главный экран (MainScreen)

### Архитектура
- Использует композицию компонентов
- Управление состоянием через React Hooks
- WebSocket соединение для real-time обновлений
- Автоматическое переподключение при разрыве соединения

### Состояния
- `selectedChat` - выбранный чат
- `chats` - список всех чатов
- `messages` - сообщения текущего чата
- `showProfile` - показ модального окна профиля
- `showCall` - показ модального окна звонка
- `theme` - текущая тема (dark/light)

## 📋 Боковая панель (Sidebar)

### Компоненты

#### Шапка
- Аватар и имя пользователя (клик открывает профиль)
- Кнопка переключения темы
- Кнопка создания канала
- Кнопка создания чата

#### Поиск
- Живой поиск по чатам
- Поиск по серверу через API
- Показ результатов в реальном времени
- Очистка результатов при пустом запросе

#### Фильтры (Папки)
- **Все** - показывает все чаты и каналы
- **Чаты** - только личные чаты
- **Каналы** - только каналы

#### Список чатов
- Аватар или иконка
- Имя чата/канала
- Последнее сообщение
- Время последнего сообщения
- Счетчик непрочитанных (badge)
- Индикатор онлайн статуса
- Подсветка активного чата

### Модальные окна

#### Новый чат
- Поиск пользователей по имени/username
- Показ списка найденных пользователей
- Создание чата при клике на пользователя

#### Новый канал
- Поле ввода названия
- Поле ввода описания (опционально)
- Валидация
- Создание канала

## 💬 Окно чата (ChatWindow)

### Шапка чата

#### Информация
- Аватар собеседника/канала
- Имя чата
- Статус:
  - Для личных чатов: "онлайн" / "не в сети" / "печатает..."
  - Для каналов: количество подписчиков

#### Действия
- Кнопка аудиозвонка
- Кнопка видеозвонка
- Кнопка поиска по чату
- Меню с дополнительными опциями

### Область сообщений

#### Отображение сообщений
- Свои сообщения справа (синий фон)
- Чужие сообщения слева (белый/темный фон)
- Характерные "хвостики" как в Telegram
- Максимальная ширина 70% экрана
- Автоматическая прокрутка к новым сообщениям

#### Структура сообщения
- Вложения (фото, документы)
- Текст сообщения
- Время отправки
- Статус прочтения (двойная галочка)
- Реакции пользователей

#### Вложения
- **Изображения**: превью с возможностью просмотра
- **Документы**: иконка файла, название, кнопка скачивания
- Поддержка множественных вложений

#### Реакции
- Показ при наведении на сообщение
- Быстрый доступ к популярным эмодзи: 👍 ❤️ 😂 😮 😢 🔥
- Группировка реакций с счетчиком
- Анимация при добавлении

### Панель ввода

#### Элементы управления
- Кнопка прикрепления файлов
- Поле ввода текста (multiline)
- Кнопка эмодзи
- Кнопка отправки / кнопка голосового сообщения

#### Функционал поля ввода
- Автоматическое изменение высоты
- Поддержка Shift+Enter для новой строки
- Enter для отправки
- Отправка индикатора "печатает..." через WebSocket

#### Emoji Picker
- Категории: Смайлы, Жесты, Сердца, Символы
- 80+ популярных эмодзи
- Быстрая вставка в текст
- Закрытие при клике вне области

#### Превью вложений
- Показ выбранных файлов перед отправкой
- Превью изображений
- Иконка для документов
- Кнопка удаления из списка
- Имя и размер файла

## 👤 Модальное окно профиля (ProfileModal)

### Режим просмотра
- Большой аватар
- Имя пользователя
- @username
- Биография (если заполнена)
- Кнопка "Редактировать профиль"
- Кнопка "Настройки"
- Кнопка "Выйти" (красная)

### Режим редактирования
- Изменение аватара (клик на камеру)
- Превью нового аватара
- Редактирование имени
- Редактирование username
- Редактирование биографии
- Кнопки "Отмена" и "Сохранить"
- Индикатор загрузки при сохранении

### Функционал
- Загрузка аватара с превью
- Валидация полей
- Отправка данных на сервер
- Обновление профиля через API
- Перезагрузка страницы после сохранения

## 📞 Модальное окно звонков (CallModal)

### Типы звонков
- Аудио звонок
- Видео звонок

### Интерфейс аудио звонка
- Большой аватар по центру с пульсацией
- Имя собеседника
- Статус звонка: "Звонок..." / таймер / "Завершен"
- Темный фон

### Интерфейс видео звонка
- Удаленное видео на весь экран
- Локальное видео в углу (Picture-in-Picture)
- Оверлей с информацией о звонке
- Таймер вызова

### Элементы управления
- **Видео**: Включить/Выключить камеру
- **Аудио**: Включить/Выключить микрофон
- **Завершить**: Красная кнопка завершения звонка
- **Настройки**: Выбор устройств
- **Полный экран**: Режим полного экрана

### WebRTC (подготовка)
- Запрос доступа к камере/микрофону
- Локальный поток медиа
- Готовая структура для peer connection
- Обработка ошибок доступа к устройствам

### Статусы звонка
1. **calling** - Исходящий звонок
2. **connected** - Активный разговор с таймером
3. **ended** - Звонок завершен

## 🔄 Real-time обновления (WebSocket)

### Обрабатываемые события

#### message
- Добавление нового сообщения в чат
- Обновление списка чатов
- Звуковое уведомление (можно добавить)

#### typing
- Показ индикатора "печатает..."
- Автоматическое скрытие через 3 секунды

#### read
- Обновление статуса прочтения
- Двойная галочка на сообщениях

#### online/offline
- Обновление статуса пользователя
- Индикатор онлайн на аватаре
- Время последнего посещения

### Отправляемые события

#### message
- Отправка нового сообщения
- Дублирование через HTTP API для надежности

#### typing
- Отправка при вводе текста
- Дебаунс 3 секунды

#### read
- Отметка сообщений как прочитанных
- Отправка при открытии чата
