# Telegram Web Clone

Полнофункциональный клон Telegram Web, построенный на React, TypeScript и Tailwind CSS.

## 🚀 Особенности

### ✅ Реализовано

- **Авторизация и Регистрация**
  - Форма входа/регистрации с валидацией
  - JWT токен аутентификация
  - Хранение сессии в localStorage

- **Профиль пользователя**
  - Редактирование имени, username и био
  - Загрузка и изменение аватара
  - Настройки профиля

- **Интерфейс мессенджера**
  - Боковая панель с поиском
  - Список чатов с последними сообщениями
  - Папки: "Все", "Чаты", "Каналы"
  - Адаптивный дизайн (mobile-friendly)

- **Чаты**
  - Создание личных чатов
  - Создание каналов
  - Отображение статуса (онлайн/офлайн)
  - Индикатор непрочитанных сообщений

- **Сообщения (Real-time через WebSocket)**
  - Отправка текстовых сообщений
  - Поддержка эмодзи с emoji picker
  - Отправка файлов (фото, документы)
  - Превью вложений
  - Реакции на сообщения
  - Индикатор "печатает..."
  - Отметки о прочтении

- **Звонки**
  - Аудио звонки
  - Видео звонки
  - UI с кнопками управления (мут, выкл. видео)
  - Подготовлено для WebRTC интеграции

- **Темы**
  - Темная тема (по умолчанию)
  - Светлая тема
  - Переключатель тем

## 🎨 Дизайн

Приложение максимально точно повторяет официальный Telegram Web:
- Фирменные цвета Telegram (#3390ec)
- Характерные закругления углов
- Иконки Font Awesome
- Плавные анимации и переходы
- Адаптивная верстка для всех устройств

## 🔧 Технологический стек

- **Frontend Framework**: React 18 + TypeScript
- **Styling**: Tailwind CSS (CDN)
- **Build Tool**: Vite
- **Icons**: Font Awesome 6
- **API Communication**: Fetch API
- **Real-time**: WebSocket
- **State Management**: React Hooks (useState, useEffect, useRef)

## 📡 Backend Integration

Приложение полностью интегрировано с backend API:

- **HTTP API**: `https://pipipupu-production.up.railway.app`
- **WebSocket**: `wss://pipipupu-production.up.railway.app`

### API Endpoints (примеры)

```
POST /auth/register - Регистрация
POST /auth/login - Вход
GET  /auth/me - Текущий пользователь

GET  /chats - Список чатов
POST /chats - Создать чат
GET  /chats/:id/messages - Сообщения чата
POST /chats/:id/messages - Отправить сообщение

POST /channels - Создать канал
GET  /search?q=... - Поиск
GET  /users/search?q=... - Поиск пользователей

POST /upload - Загрузка файлов
POST /upload/avatar - Загрузка аватара

POST /messages/:id/reactions - Добавить реакцию
```

### WebSocket Events

```javascript
// Отправка
ws.send({ type: 'message', data: { chatId, content, attachments } })
ws.send({ type: 'typing', data: { chatId } })
ws.send({ type: 'read', data: { chatId, messageId } })

// Получение
{ type: 'message', data: { id, chatId, userId, content, ... } }
{ type: 'typing', data: { chatId, userId } }
{ type: 'online', data: { userId } }
{ type: 'offline', data: { userId } }
```

## 📱 Функционал по экранам

### Экран авторизации
- Переключение между входом и регистрацией
- Валидация полей
- Обработка ошибок
- Автоматический вход при наличии токена

### Главный экран
- **Боковая панель**:
  - Аватар и имя пользователя
  - Поиск по чатам и контактам
  - Фильтры (Все/Чаты/Каналы)
  - Кнопки создания чата/канала
  - Переключатель темы

- **Окно чата**:
  - Шапка с информацией о чате
  - История сообщений
  - Поле ввода с поддержкой multiline
  - Emoji picker
  - Прикрепление файлов
  - Кнопки звонков
  - Реакции на сообщения

### Модальные окна
- **Профиль**: Просмотр и редактирование
- **Звонки**: Интерфейс аудио/видео звонков
- **Новый чат**: Поиск пользователей
- **Новый канал**: Создание канала
